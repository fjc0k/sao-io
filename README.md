# sao-io

> my extraordinary SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao io my-project
```

### From git

```bash
sao jay fong/sao-io my-project
```
